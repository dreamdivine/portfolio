AT SEED WORLD is a clone of a seed website called rareseeds. I frequently purchase seeds from this website for my garden and I wanted to try to replicate it. This app is created mainly using Ruby on Rails, React and Javascript in order to generate a true single page reactive web app experience. Recreating the website that I use so often really helped me challenge myself and appreciate the work that goes into making an app even more.
